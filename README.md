# Event Registration System

## Description

The Event Registration System is a web-based application that allows users to register for events and enables administrators to manage participant responses. The system stores registration details in a database and categorizes users as responded or unresponded based on their status and send a confirmation mail to the responded users and reminder mail to the unresponded users.

## Features

- User event registration
- User dashboard to register for an event
- Admin dashboard to view registrations
- Categorization of users as Responded and Unresponded
- sending confirmation and reminder mail to the responded and unresponded users
- Data stored and retrieved from MySQL database
- Dynamic data display using Servlets

## Technologies Used

- Frontend: HTML
- Backend: Java, Servlets
- Database: MySQL
- Server: Apache Tomcat
- IDE: Eclipse
